1. **Ruta para recibir datos de control**:
   - **URL**: `/control`
   - **Método**: POST
   - **Descripción**: Esta ruta recibe los datos de control enviados por el control a través de WiFi. Los valores para cada grado de libertad (ángulos de articulación) se incluyen en el cuerpo de la solicitud.

2. **Ruta para obtener datos almacenados**:
   - **URL**: `/datos`
   - **Método**: GET
   - **Descripción**: La aplicación se conecta a esta ruta para obtener los datos almacenados en el servidor. El servidor envía los datos previamente recibidos del control.

3. **Ruta para enviar datos al robot**:
   - **URL**: `/robot`
   - **Método**: POST
   - **Descripción**: La aplicación envía los datos al robot a través de esta ruta. El servidor toma los datos recibidos y los envía al robot a través del puerto serie.

4. **Ruta de estado del servidor**:
   - **URL**: `/estado`
   - **Método**: GET
   - **Descripción**: Proporciona información sobre el estado actual del servidor (por ejemplo, si está en línea o desconectado).
